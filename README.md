# 家庭科テスト対策アプリ

家庭科のテスト範囲を、4択クイズで練習できる静的 Web アプリです。スマホで片手操作しやすいように、大きなボタンと読みやすいカード型レイアウトにしています。

## 主な機能

- 1問ずつ問題を表示
- 選択肢をタップすると正誤判定
- 正解と短い解説を表示
- 最後に正答数と正答率を表示
- 間違えた問題を `localStorage` に保存
- 間違えた問題だけを復習
- ランダム出題モード
- 画像付き問題に対応
- 包丁、基本切り、鍋の水量は自作 SVG イラスト付き

## 使い方

1. `index.html` をブラウザで開きます。
2. 「はじめる」を押すと通常モードで出題されます。
3. 「ランダム」をオンにすると、問題順が毎回シャッフルされます。
4. 選択肢を選ぶと、正解・不正解と解説が表示されます。
5. 最後に結果が表示されます。
6. 間違えた問題がある場合は「間違えた問題だけ復習」から再挑戦できます。

## 問題の追加方法

問題は `questions.js` の `QUESTIONS` 配列に追加します。

```js
{
  id: "seasoning-001",
  category: "調味料",
  important: true,
  question: "大さじ1は何ml？",
  choices: ["5ml", "10ml", "15ml", "20ml"],
  answer: "15ml",
  explanation: "大さじ1は15mlで、小さじ3杯分です。",
  image: "",
  imageAlt: ""
}
```

画像付き問題にする場合は、`image` に画像パス、`imageAlt` に代替テキストを入れます。画像がない問題では空文字のままで動作します。

## GitHub Pages 公開方法

1. GitHub にリポジトリを作成します。
2. このフォルダのファイルを push します。
3. GitHub のリポジトリ画面で `Settings` を開きます。
4. `Pages` を開きます。
5. `Build and deployment` の `Source` を `Deploy from a branch` にします。
6. `Branch` を `main`、フォルダを `/root` にして保存します。
7. 表示された GitHub Pages の URL にアクセスします。

このアプリは HTML / CSS / JavaScript のみで動くため、ビルド作業は不要です。
